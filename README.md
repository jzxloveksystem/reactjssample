> react + redux
- component 밖에서 store라는 객체 내부에 상태를 담고 관리한다.
- 상태에 어떤 변화를 일으켜야 하면, action을 store에 전달한다. 

> 개념 : 
- Action 액션 
- Action Creator 액션 생성함수 
- Reducer 리듀서 
- Store 스토어 => 상태 담는다. 
- dispatch 디스패치 => store에 action을 전달하는 과정 
- subscribe 구독
